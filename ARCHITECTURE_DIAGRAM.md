## What Each Component Does

### Browser (INFINITY App)
- **Runs**: React SPA served from GitHub Pages
- **Stores**: Everything in browser storage (tokens, pages, etc.)
- **Cannot**: Write files to GitHub, create commits, push changes
- **Can**: Generate HTML, download files, show previews

### Your Computer
- **Has**: Downloaded HTML and JSON files
- **Does**: Manual git operations (add, commit, push)
- **Bridges**: Gap between browser storage and GitHub repo

### GitHub Repository
- **Contains**: Source code, workflows, manually committed pages
- **Triggers**: Workflow on every push to main
- **Stores**: All published content

### GitHub Actions
- **Runs**: pages.yml workflow automatically
- **Builds**: Spark app from source
- **Deploys**: Everything to GitHub Pages
- **Cannot**: Access browser storage, generate pages automatically

### GitHub Pages
- **Serves**: Built Spark app at root URL
- **Serves**: Any manually committed page folders
- **Updates**: On every workflow completion
